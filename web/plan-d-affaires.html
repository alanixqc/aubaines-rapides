2 Engine de Recettes — L'IA au Cœur

Mois 2-4 · Coût: ~500-1000$ (API IA) · Revenu estimé: 100-300$/mois

🤖

Générateur de Recettes IA

Script Python qui prend les meilleurs deals de la semaine → LLM génère 3 recettes originales. Chaque recette inclut: liste d'épicerie, temps, coût/portion, macros.

Automatisé
🛒

Liste d'Épicerie Dynamique

Les recettes listent les ingrédients exacts en vente cette semaine. Lien direct vers chaque item sur Flipp. L'utilisateur voit: « 3 recettes = 23.47$ d'épicerie ».

15h dev
📹

Vidéos Recette Automatiques

Script génère des vidéos TikTok/Reels à partir des recettes: images générées + overlay texte + musique libre de droits. Montage auto avec FFmpeg + Pillow.

Script dev
🍳

Affiliation Ustensiles Ciblée

Chaque recette recommande 3 outils Amazon: « Parfait avec une friteuse à air », « Un bon couteau Chef fait toute la différence ». Liens profonds Amazon.

✧ 4-10%

🎯 Objectif: 3 nouvelles recettes/semaine, 500-1000 visiteurs/mois, 50 clics Amazon/semaine. 💰 Potentiel: 100-300$/mois (affiliation recettes + ustensiles).

📅 Timeline de Lancement

Les 12 prochaines semaines — actions concrètes, pas de bullshit

✅ Semaine 1-2 — Site Web Public

Maintenant

Héberger le rapport HTML sur GitHub Pages. Ajouter formulaire newsletter (MailerLite gratuit). Mettre en place Google Analytics et Search Console. Ajouter un blog minimal.

📝 Semaine 3-4 — Engine Recettes IA

~4h de dev

Script Python qui prend les 3 meilleurs deals → appelle LLM (DeepSeek) → génère recette structurée (nom, ingrédients, étapes, coût/portion). Stocker dans SQLite. Publier sur site. Première intégration liens Amazon.

📰 Semaine 4-6 — Newsletter + Growth

~6h total

Template email hebdomadaire. Automatiser l'envoi avec MailerLite API. Call-to-action: abonnement premium. Popup d'inscription sur le site. Land page pour attirer les abonnés via Facebook/Reddit Québec.

🎬 Semaine 5-8 — Contenu TikTok/YouTube

~3h/semaine

Vidéos « Top 3 deals de la semaine » format 60s. Lien en bio vers le site. Vidéo recette « 4 portions à 2.50$ avec les deals de cette semaine ». Publier 2-3/semaine.

💎 Semaine 6-10 — Abonnement Premium

~10h de dev

Système d'authentification simple (Google OAuth ou email). Paiement Stripe. Pages membres: historique graphique, alertes prix, export CSV. Stripe prend 2.9%+0.30$, toi tu gardes ~3.50$ sur 3.99$.

📊 Semaine 8-12 — Plan Repas Pro

~8h de dev + IA

L'utilisateur entre budget + nb de personnes + restrictions → IA génère plan repas 7 jours basé sur les deals courants. Option génération PDF. Marque blanche possible pour influenceurs food.

💡 Pourquoi Ça Va Marcher

Les 3 piliers qui rendent ce projet unique

🎯

1. Besoin Réel — Québec

Les prix de l'épicerie explosent. Les gens veulent économiser. Il n'existe AUCUN service québécois qui combine deals de viande + recettes + planification. Tes concurrents: Flipp (générique, sans analyse) et des blogueuses recettes (sans deals).

🤖

2. Automatisation Extrême

Scraper + IA + génération contenu = tout tourne tout seul. Pas de rédacteur à payer. Pas de montage vidéo manuel. Tu bosses sur la machine, pas dans la machine. 2-3h/semaine max une fois lancé.

🔄

3. Boucle Vertueuse

Plus tu as de données → meilleures sont tes recettes → plus de trafic → plus d'abonnés → plus de revenus → meilleure infrastructure → meilleures données. Chaque mois, la machine s'améliore d'elle-même.
